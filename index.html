HOW IT WORKS

01
📷
Webcam input
OpenCV reads your live webcam feed frame by frame.
02
🦴
Pose detection
MediaPipe Pose Landmarker tracks your body joints in real-time.
03
📐
Form analysis
Joint angles are calculated and checked against correct form thresholds.
04
💬
Visual feedback
Rep count and form cues are overlaid directly on the live video.
// currently supported

EXERCISES

SETUP

1
Clone the repo
git clone https://github.com/himanshu-tw/exercise-form-checker cd exercise-form-checker
2
Create a virtual environment
python -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate
Requires Python 3.8+
3
Install dependencies
pip install -r requirements.txt pip install keyboard
4
Download the pose model
wget https://storage.googleapis.com/mediapipe-models/pose_landmarker/pose_landmarker_lite/float16/pose_landmarker_lite.task -O pose_landmarker.task
Model file goes in the project root directory.
5
Run
python main.py
Press q to quit.
